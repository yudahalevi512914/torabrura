# תורה ברורה

אתר תדמית ותרומות ראשוני לעמותה העוסקת בלימוד תורה בארץ ישראל.

## תכולה ראשונית

- כותרת ראשית: תורה ברורה
- כותרת משנה: החלק שלך בתורת ארץ ישראל
- אזורי תרומה: כוללים, בניית יישובים, הוצאת ספרים, חיזוק העלייה לארץ
- עיצוב RTL בעברית בהשראת עמוד בחירת התרומה של עטרת שלמה

## הפעלה מקומית

אפשר לפתוח את `index.html` ישירות בדפדפן.
